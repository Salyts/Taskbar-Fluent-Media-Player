# Taskbar Fluent Media Player

**Taskbar Fluent Media Player** — это мод для Windhawk, который интегрирует современный медиаплеер с Fluent Design прямо в панель задач Windows 11. Он позволяет управлять музыкой и просматривать информацию о треке, не отвлекаясь от работы.

### [> Документация на английском языке <](https://github.com/Salyts/Taskbar-Fluent-Media-Player)

## Установка

1. Установите [Windhawk](https://windhawk.net/).
2. Откройте **Windhawk** → **Explore** → **Search**.
3. Введите **Taskbar Fluent Media Player** в поиске.
4. В результатах выберите [Taskbar Fluent Media Player](https://windhawk.net/mods/taskbar-fluent-media-player).
5. Нажмите кнопку "Install".

## Основные возможности

* **Глубокая интеграция —** размещение плеера в области трея (слева или справа от часов), рядом с системными значками или по центру панели задач.
* **Эффекты Fluent UI —** полная поддержка нативных материалов Windows 11, включая **Acrylic**, **Mica** и **Mica Alt**.
* **Адаптивный дизайн —** плеер может автоматически подстраивать цвета под обложку альбома или использовать системный акцентный цвет.
* **Полное управление медиа —** кнопки Play/Pause, переключение треков, перемотка на 5 секунд, Shuffle и Repeat.
* **Умное поведение —** автоматическое скрытие плеера, когда музыка не воспроизводится, при переходе в полноэкранный режим или после периода бездействия.
* **Аудиовизуализатор —** Визуализатор спектра в реальном времени (петля WASAPI + БПФ) с несколькими формами полос (стерео, гора, зеркало, волна, дыхание), цветовыми режимами (сплошной, динамический альбом, градиент, акриловый, акцент), предустановками эквалайзера, вертикальной привязкой и размещением по сторонам.

## Расширенная настройка

* **Внешний вид —** настройка шрифтов (Segoe UI, Aptos и др.), размеров текста, отступов и прозрачности элементов.
* **Обложка альбома —** настройка скругления углов, наложение иконки приложения-источника и эффекты паузы.
* **Взаимодействие мышью —** назначение пользовательских действий (управление громкостью, перемотка трека, переключение сессий) на колесо мыши, одиночные и двойные клики по различным частям плеера.
